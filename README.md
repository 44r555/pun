# pun
Docker image for Python-uWSGI-Nginx

This image is based on python:3.6-slim, with minimalistic uwsgi install and nginx

## How to build docker image
```
docker build -t pun .
```


## Build image with only Python-uWSGI
```
docker build -t pu pu
```
